# Azure Data Factory End-to-End ETL Pipeline Project

![Architecture](https://github.com/RashWadhwa/adf-etl-project/blob/main/Session-5/images5/maindiagram.png)

This repository contains the source control and documentation for a comprehensive ETL pipeline developed as part of the **Master Azure Certificate** from **AzureLib**.

## Project Overview
The project demonstrates a full data lifecycle, moving data from raw sources into structured cloud storage and relational databases using **Azure Data Factory (ADF)**.

### Key Achievements:
* **Source Systems**: Orchestrated data ingestion from a sample SQL Database.
* **Multi-Format Processing**: Created and managed datasets in **CSV, JSON, and Parquet** formats.
* **Storage Architecture**: Implemented data landing zones across **Azure Data Lake Storage (ADLS) Gen2**, **Azure Blob Storage**, and **Azure SQL Database**.
* **Pipeline Automation**: Built end-to-end ADF pipelines to ensure data integrity and automated movement.

### Next Steps:
* Currently integrating **Azure Databricks** to perform advanced transformations and big data processing.
